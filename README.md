# Student Management System (Spring Boot + Postman)

A simple Student Management System built with Spring Boot.  
It supports full CRUD operations (Create, Read, Update, Delete)  
using REST APIs tested in Postman.

## Tech Stack
- Java 21
- Spring Boot 3.2.5
- Spring Data JPA
- H2 Database
- Postman
